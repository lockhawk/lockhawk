Initial release: fast, free, accurate npm dependency vulnerability scanner powered by
OSV.dev. Scans `package-lock.json`, `yarn.lock`, and `pnpm-lock.yaml`; offline/online/auto
sources with on-disk caching and fail-open resilience; CVSS v3 and v4 scoring; table, JSON,
SARIF, and self-contained HTML reports plus an interactive `serve` dashboard; and a GitHub
Action for CI/CD.
